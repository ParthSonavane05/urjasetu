# ⚡ Urja Setu

## 🌍 Problem Statement

Energy consumption is often inefficient due to lack of awareness, real-time monitoring, and actionable insights. Users are unable to track, optimize, and reduce their energy usage effectively.

---

## 💡 Solution

Urja Setu is a smart energy management platform that helps users monitor, analyze, and optimize their energy consumption using real-time data and intelligent insights.

---

## 🚀 Features

- 📊 Real-time energy monitoring  
- 🤖 Smart recommendations for energy saving  
- 📱 Clean and user-friendly UI  
- 🔐 Secure data handling using Firebase  
- 🌐 Backend integration with Flask APIs  

---

## 🛠 Tech Stack

- **Frontend:** Flutter  
- **Backend:** Flask  
- **Database & Auth:** Firebase  
- **Version Control:** GitHub  

---

## 🏗 Architecture Overview

The system follows a modular architecture:

- Flutter App → User Interface  
- Firebase → Authentication & Database  
- Flask API → Business Logic & Processing

🚀 How to Clone and Run Urja Setu Project

✅ STEP 1: Install Required Tools

Make sure these are installed:
	•	Flutter SDK
	•	Git

Check:
```bash
flutter doctor
```

📥 STEP 2: Clone the Project

Open Terminal and run:
```bash
git clone https://github.com/ParthSonavane05/urjasetu.git
```

📁 STEP 3: Go Inside Project Folder
```bash
cd urjasetu
```

📦 STEP 4: Install Dependencies
```bash
flutter pub get
```

▶️ STEP 5: Run the App
```bash
flutter run
```

📱 STEP 6: Select Device (if needed)

If no device is connected:
	•	Open Android Emulator
OR
	•	Connect your phone

Then run again:
```bash
flutter run
```

⚠️ If Any Error Comes

Run this:
```bash
flutter clean
flutter pub get
flutter run
```
